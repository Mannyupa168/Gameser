# Gameser
Gameser es una tienda online que vende videojuegos para PC, Playstation, Nintendo y Xbox. Aquí todo el mundo puede encontrar lo que necesita a costes competitivos y estar al tanto de todos los eventos y novedades del mundo del gaming. Mi principal objetivo era crear una tienda online funcional para juegos de PC.

Cree un sitio web elegante con temas oscuros y acentos brillantes. Colocando las secciones más importantes en la página de inicio y haciendo que el camino de compra sea lo más claro y corto posible para el usuario.

El principio de compra es similar al de Epic Games. Un cliente compra un juego y el servicio envía un enlace para descargar el juego por correo electrónico. Cuando el cliente hace clic en el enlace, va a la cuenta personal y descarga el juego en su ordenador utilizando el descargador.
